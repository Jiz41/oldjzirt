# 華耀天輪 真・自在律

競輪レースの出走データをもとに、独自のシミュレーションロジックで予想を行う個人開発ツールです。
勝敗を保証するものではありません。あくまで個人参考用にご利用ください。

A personal tool that simulates keirin race predictions based on entry data using original logic.
This does not guarantee any outcome. For personal reference only.

---

## 使い方 / How to Use

**手動入力の場合 / Manual Input**
1. 出走表を参照しながら、各項目を手動で入力します
2. シミュレーション結果をもとに買い目を検討してください

1. Refer to the race entry sheet and fill in all fields manually.
2. Review the simulation results to consider your bets.

**自動入力の場合 / Auto Input**
1.「レース情報自動取得」ボタンを押します
2. 本日開催中のレース一覧が表示されます
3. レースを選択するとデータが自動で読み込まれます
4. 不足している項目は手動で入力します
5. シミュレーション結果をもとに買い目を検討してください

1. Press the "レース情報自動取得" (Auto Fetch) button.
2. A list of today's races will appear.
3. Select a race and the data will load automatically.
4. Fill in any remaining fields manually.
5. Review the simulation results to consider your bets.
