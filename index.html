⚙️ ツールの全体構成と基本操作

このツールは、入力されたレース情報と選手データに基づき、複数の展開パターンをシミュレーションし、独自ロジックで評価した予想結果を提供するモバイル向け競輪予想支援ツールです。以下の手順に従って入力を行ってください。

1. レース基本情報の入力（土台を作る）

計算のベースとなる係数と、バンク固有の特性を決定します。

項目 入力内容・例 解説：システムへの影響
級班 ガールズ / S級 / A級 / A級チャレンジ 選択した級班に応じて、計算に使用される計算係数が最適化されます。
バンク名 開催バンク名を選択 バンク特有の展開傾向（逃げ・差し有利等）が自動抽出され、結果エリアに反映されます。

2. 選手個別データの入力（精度を高める）

出走7名の詳細ステータスを入力します。各行の項目を左から順に埋めてください。

項目 内容・選択 解説：入力の目的
地元 / 欠場 チェックボックス 地元利の個別補正、または欠場選手の計算除外を処理します。
脚質 / 得点 逃先・差マ・捲り / 競走得点 選手の基本戦術と機力を評価し、シミュレーションの基礎とします。
W印 / 3走 ◎〇△✕等 / 直近着順(111) 専門紙による客観評価と、近況のリズム（調子）を指数に加味します。
S 1位 / B 1位 ラジオボタンで1名ずつ指定 S（スタンディング）とB（バック）の1着候補を特定し、展開の核を決定します。

3. 展開（ライン構成）の入力：記述ルール

「全体並び」エリアに、ライン構成と補正の最終判断を入力します。

状態・項目 書き方（例） 解説：計算への反映
基本並び 123,45,67 ラインをカンマで区切ります。結果エリアにライン強度がグラデーションで表示されます。
競り発生 1(3)2, 45 カッコ内の選手が競りを挑むと認識。強度の減衰や展開の紛れを反映させます。
🎌地元スイッチ チェックをON 地元選手への恩義や番組配慮を加味した地元利補正を全体計算に適用します。

4. 予想の実行とアウトプット（結果を受け取る）

出力項目 内容と活用方法
☀️ 晴天令 安定推奨：最も安定し、堅い決着が予想される推奨買い目です。
⛈️ 荒天令 高配当狙い：波乱の可能性を重視し、紛れを含んだ推奨買い目です。
天雲指数 独自ロジックによる期待値。全体のパワーバランスを数値化します。
処理ログ ツール内部でどのような計算ステップが行われたかを確認できます。
