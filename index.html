⚙️ ツールの全体構成と基本操作

このツールは、入力されたレース情報と選手データに基づき、複数の展開パターンをシミュレーションし、独自ロジックで評価した予想結果を提供するモバイル向け競輪予想支援ツールです。

1. レース基本情報の入力（土台を作る）

項目 入力内容・例 解説：システムへの影響
級班 ガールズ / S級 / A級等 級班に応じた計算係数の最適化。
バンク名 開催場所を選択 バンク固有の展開傾向（逃げ・差し等）を自動抽出。
風向・風速 8方位 / 0.5m単位 V8.0 捌風旋炉 枢：風によるライン位置別のスタミナ損耗（kururu係数）を演算。

※風情報は tenki.jp や 風競輪.info を参照。無風や屋内時は「無風/屋内」を選択してください。

2. 選手個別データの入力（精度を高める）

項目 内容・選択 解説：入力の目的
地元 / 欠場 チェックボックス 地元利の個別補正、および欠場選手の計算除外。
脚質 / 得点 逃先・差マ・捲り / 競走得点 選手の基本戦術と機力の評価。
W印 / 3走 ◎〇△✕等 / 直近着順 専門紙評価と近況のリズムを指数に加味。
S 1位 / B 1位 ラジオボタン指定 スタンディングとバックの1着候補から展開の核を決定。

3. 展開（ライン構成）の入力

状態・項目 書き方（例） 解説：計算への反映
基本並び 123,45,67 ラインをカンマ区切りで入力。強度をグラデーション表示。
競り発生 1(3)2, 45 カッコ内を競り選手と認識。展開の紛れを反映。
🎌地元スイッチ チェックON 地元勢への恩義や番組構成を加味した全体補正。

4. 予想の実行とアウトプット

出力項目 内容と活用方法
☀️ 晴天令 安定推奨：最も支持率と物理条件が安定した推奨買い目。
⛈️ 荒天令 高配当狙い：自滅消耗や風の影響による波乱狙い。
天雲指数 独自ロジックによる期待値。全体のパワーバランスを数値化。
処理ログ 物理演算や各選手の補正ステップの確認。
