⚙️ ツールの全体構成と基本操作

このツールは、入力されたレース情報と選手データに基づき、複数の展開パターンをシミュレーションし、独自ロジックで評価した予想結果を提供するモバイル向け競輪予想支援ツールです。

1. レース基本情報の入力（土台を作る）

項目 入力内容・例 解説：システムへの影響
級班 ガールズ / S級 / A級等 級班に応じた計算係数の最適化。
バンク名 開催場所を選択 バンク固有の展開傾向（逃げ・差し等）を自動抽出。
風向・風速 8方位 / 0.5m単位 V8.0 捌風旋炉 枢：風によるライン位置別のスタミナ損耗（kururu係数）を演算。

※風情報は tenki.jp や 風競輪.info を参照。無風や屋内時は「無風/屋内」を選択してください。

2. 選手個別データの入力（精度を高める）

項目 内容・選択 解説：入力の目的
👑 GCボタン チェックボックス 若手の潜在能力補正。得点以上の実力を持つ選手を、上位層の期待値へ引き上げます。
※過度な補正がかかるため、使用は慎重に判断してください。本来の予想精度を大きく損なう恐れがあります。
地元 / 欠場 チェックボックス 地元利の個別補正、および欠場選手の計算除外。
※システムによる自動判別は行いません。出走表等で確認し、手動でチェックを入れてください。
脚質 / 得点 逃先・差マ・捲り / 競走得点 選手の基本戦術と機力の評価。決まり手の中で最も高い値のものを指定してください。
※数値が同数の場合は、今回の役割・位置（自力か番手か）を優先して選択してください。
W印 / 3走 ◎〇△等 / 3走成績 専門紙評価と近況のリズムを反映。
新しい順に、3走の成績を入力してください。

S 1位 / B 1位 ラジオボタン指定 スタンディングとバックの1着候補から展開の核を決定。

4. 予想の実行とアウトプット

出力項目 内容と活用方法
天雲指数 独自ロジックによる安定値：七曜院パラドキサ環による単次元的解析。
☀️ 晴天令 最も支持率と物理条件が安定した推奨買い目。
⛈️ 荒天令 自滅消耗や風の影響による波乱狙い。
🌌 赤口呑縁 1,465の並行世界シミュレーション：五更斎アメンティア清音による多次元展開解析。
• 出現出目：最も頻度の高い決着パターンTOP3を表示
• 因果分岐：ライン崩壊や波乱イベント発生時の出目傾向
• 選手の生存濃度：各選手の1着〜3着入線回数と圏外率を集計
※級班ごとに異なる「世界線の重み付け」により、安定志向と波乱志向のバランスを調整
処理ログ 物理演算や各選手の補正ステップの確認。

5. 赤口呑縁（しゃっこうどんぺり）について

偽典『赤口呑縁』は、1,465通りの並行世界をシミュレーションし、展開の「揺らぎ」を可視化する解析機構です。

要素 説明
世界線の構成 通常展開（晴天令ベース）と波乱展開（荒天令ベース）を級班ごとの比率で混合。
例：S級 = 879通常 + 586波乱 / A級 = 586通常 + 879波乱
”ゆらぎ”の詳細 ライン微崩壊・早駆け・風煽り・競り乱れ・落車・ダークホース・新人暴走・捲り自滅など、
展開を変動させる8種のイベントをランダム発生させる。
出現出目 1,465回の試行で最も多く出現した決着パターンを、頻度順にTOP3表示。
※頻度が高いほど「その決着に収束しやすい」ことを示す。
因果分岐 特定のカオスイベント（ライン崩壊など）が発生した世界線のみを抽出し、
その条件下で最も出やすい決着パターンを分析。
※「波乱時の保険」として活用可能。
生存濃度 各選手が1着・2着・3着・圏外にそれぞれ何回入線したかを集計。
※着順分布を見ることで、選手の「安定性」や「ヒモ適性」が判断できる。

🎯 赤口呑縁の活用方法

  • 本命固め：出現出目TOP1が晴天令と一致 → 安定した展開
  • ヒモ選択：生存濃度で2着・3着の入線回数が多い選手を狙う
  • 波乱対策：因果分岐の出目を保険として買う
  • 買い目の絞り込み：頻度が極端に低い決着（10回未満など）を除外

※赤口呑縁は疑似統計的な傾向を示すものであり、実際のレース結果を保証するものではありません。
※計算には数秒かかる場合があります（1,465回のシミュレーション実行のため）。
